# 📰 LatestNews

A modern news web application built using HTML, CSS, and JavaScript.

## ✨ Features

- 🌙 Dark & Light Theme
- 🔍 Search News
- 📰 Category Filters
- ♾ Infinite Scrolling
- 📱 Responsive Design
- ⚡ Fast UI
- 🖼 Default Image Support

## 🌐 API

This project uses the **NewsAPI** to fetch the latest news articles from various trusted sources in real time.

## 🛠 Technologies

- HTML
- CSS
- JavaScript
- NewsAPI
